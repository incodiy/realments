Build Dynamic Forms with Laravel & React

Realments is a powerful form builder package that allows you to create dynamic form elements with just a few lines of code in your Laravel controller.

Get Started GitHub 
// In your Laravel controller
$form = app('realments');

$form->open(['action' => route('users.store')]);

$form->text('name', 'John Doe')
    ->rules('required|max:255');

$form->email('email')
    ->rules('required|email');

$form->select('country', [
    'Select a country',
    'usa' => 'United States',
    'canada' => 'Canada',
    'uk' => 'United Kingdom'
]);

$form->close('Submit');

return view('users.create', [
    'form' => $form->render()
]);

Installation

Get up and running in minutes

1

Install via Composer

composer require incodiy/realments
2

Publish Assets

php artisan vendor:publish --provider="Incodiy\Realments\Providers\RealmentsServiceProvider" --tag="realments-assets"
3

Publish Configuration (Optional)

php artisan vendor:publish --provider="Incodiy\Realments\Providers\RealmentsServiceProvider" --tag="realments-config"
4

Add Facade to config/app.php (Laravel 10 only)

'aliases' => [
    // ...
    'Realments' => Incodiy\Realments\Facades\Realments::class,
],

Basic Usage

Start building forms in minutes

In Your Controller

public function create()
{
    $form = app('realments');
    
    // Open form
    $form->open([
        'action' => route('users.store'),
        'method' => 'POST',
        'files' => true, // If you need file uploads
        'css_framework' => 'bootstrap', // Options: bootstrap, tailwind, bulma
        'theme_mode' => 'light' // Options: light, dark
    ]);
    
    // Add form elements
    $form->text('name', 'John Doe', [
        'placeholder' => 'Enter your name',
        'required' => true
    ])->rules('required|max:255');
    
    $form->email('email', 'john@example.com')
        ->rules('required|email');
    
    $form->select('country', [
        'Select a country',
        'usa' => 'United States',
        'canada' => 'Canada',
        'uk' => 'United Kingdom'
    ]);
    
    // Close form with submit button
    $form->close('Submit');
    
    // Render the form
    return view('users.create', [
        'form' => $form->render()
    ]);
}

In Your Blade Template

<!-- resources/views/users/create.blade.php -->

Form Components

Explore all available form elements

Basic Inputs Select Inputs Checkbox & Radio Date & Time File Uploads Advanced Inputs

Basic Inputs

// Text input
$form->text('name', 'John Doe', [
    'placeholder' => 'Enter your name',
    'class' => 'custom-class'
]);

// Email input
$form->email('email', 'john@example.com');

// Password input
$form->password('password');

// Number input
$form->number('age', 25, [
    'min' => 18,
    'max' => 100,
    'step' => 1
]);

// Textarea
$form->textarea('description', 'Lorem ipsum dolor sit amet', [
    'rows' => 5
]);

Select Inputs

// Basic select
$form->select('country', [
    'Select a country',
    'usa' => 'United States',
    'canada' => 'Canada',
    'uk' => 'United Kingdom'
]);

// Multi-select
$form->select('countries', [
    'Select countries',
    'usa' => 'United States',
    'canada' => 'Canada',
    'uk' => 'United Kingdom'
], [
    'multiselect' => true,
    'selected' => ['usa', 'uk']
]);

// Select with add button
$form->select('skills', [
    'Select a skill',
    'php' => 'PHP',
    'js' => 'JavaScript',
    'python' => 'Python'
], [
    'add_button' => true,
    'max_additions' => 5,
    'button_position' => 'right',
    'button_text' => 'Add Skill'
]);

Checkbox & Radio

// Checkbox
$form->checkbox('agree_terms', '1', true, [
    'label' => 'I agree to the terms and conditions'
]);

// Radio buttons
$form->radio('gender', [
    'male' => 'Male',
    'female' => 'Female',
    'other' => 'Other'
], 'male');

// Switch
$form->switch('notifications', '1', true, [
    'label' => 'Enable notifications'
]);

Date & Time

// Date input
$form->date('birth_date', '1990-01-01');

// Time input
$form->time('meeting_time', '14:30');

// DateTime input
$form->datetime('appointment', '2023-05-15T14:30');

// Date range
$form->dateRange('vacation', ['2023-06-01', '2023-06-15']);

File Uploads

// Basic file upload
$form->file('document');

// Image upload with thumbnail
$form->file('profile_picture', [
    'thumbnail' => true,
    'thumbnail_size' => 150,
    'thumbnail_position' => 'top',
    'accept' => 'image/*'
]);

Advanced Inputs

// Rich text editor
$form->richText('content', '
Hello World
', [
    'editor' => 'tinymce' // Options: tinymce, ckeditor, quill
]);

// Color picker
$form->color('theme_color', '#3490dc');

// Range slider
$form->range('rating', 5, [
    'min' => 0,
    'max' => 10
]);

// Tags input
$form->tags('keywords', ['laravel', 'react']);

// Autocomplete
$form->autocomplete('city', 'New York', [
    'New York',
    'Los Angeles',
    'Chicago',
    'Houston',
    'Phoenix'
]);

// Captcha
$form->captcha();

API Reference

Complete documentation of all available methods

Method Parameters Description
open() array $attributes = [] Opens a form with the specified attributes.
close() string|null $submitText = 'Submit', array $attributes = [] Closes the form and optionally adds a submit button.
render() None Renders the form and returns the HTML.

Method Parameters Description
text() string $name, string $value = null, array $attributes = [] Creates a text input field.
textarea() string $name, string $value = null, array $attributes = [] Creates a textarea field.
select() string $name, array $values, array $attributes = [] Creates a select dropdown field.
checkbox() string $name, string $value = '1', bool $checked = false, array $attributes = [] Creates a checkbox field.
radio() string $name, array $options, string $checked = null, array $attributes = [] Creates radio button fields.

Method Parameters Description
rules() array|string $rules, array $messages = [] Sets validation rules for the current element.
